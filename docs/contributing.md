# Milestones

[Milestone](https://github.com/svandragt/lamb/milestones) goals can be generated using the following commands:
```
# Shell
$ gh issue list --milestone "0.6.0" --json title,body | copy

# ChatGPT
> I got the following json and I want to write a milestone goals for this: $JSON
```
